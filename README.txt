■操作方法

キーボード操作
・方向キー：キャラクターの移動
・Z/Enter：決定、調べる
・X/Esc：キャンセル、メニュー表示
・Shift：ダッシュ

マウス操作
・左クリック：決定、移動
・右クリック：キャンセル、メニュー表示

特殊な操作
・決定キー、左クリックの長押し：イベントの早送り
・Ctrl：メッセージスキップ
・A：メッセージの自動送り
・右クリック、Shift：メッセージウィンドウの一時消去
・Tab：バックログの表示
・Qキー：時間待機（ＣＨＡＰＴＥＲ：２以降に解禁）
・Ｅキー：地図メニュー
